# gemeente-m2m-inburgering
Het programma Veranderopgave Inburgering (VOI) van het inburgeringsstelsel WI2021 bevat een onderdeel voor het realiseren van een geautomatiseerde gegevensuitwisselingen tussen DUO en de ketenpartners in de inburgeringsketen. De informatie in deze repository is opgesteld voor de M2M gegevensuitwisseling tussen DUO en een gemeente.
